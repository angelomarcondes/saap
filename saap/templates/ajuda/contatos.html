Contatos


A primeira e principal funcionalidade do SAAP é a gestão de contatos. É possível cadastrar diversos dados de um contato, como dados básicos, documentos, endereços, telefones, e-mails, dados educacionais e profissionais, filiação partidária, parentescos, e outros. Além disto, o SAAP também permite criar grupos, bem como fazer uma pesquisa completa no cadastro.

O sistema também permite a exportação de contatos para planilha, seja para armazenamento, ou para manipulação de dados. No futuro, será possível fazer o processo inverso: importar a partir de uma planilha.
